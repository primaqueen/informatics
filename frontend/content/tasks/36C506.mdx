Автоматическая камера производит растровые изображения размером 400×512 пикселей. Для кодирования цвета каждого пикселя используется одинаковое количество бит, коды пикселей записываются в файл один за другим без промежутков. Объём файла с изображением не может превышать 320 Кбайт без учёта размера заголовка файла. Какое максимальное количество цветов можно использовать в палитре?
