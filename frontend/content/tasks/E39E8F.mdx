В языке запросов поискового сервера для обозначения логической 
операции «ИЛИ» используется символ «|», а для обозначения логической операции «И» – символ «&».

В таблице приведены запросы и количество найденных по ним страниц некоторого сегмента сети Интернет.


<table> <tr> <td> <p><b>Запрос</b></p> </td> <td> <p><b>Найдено страниц</b></p> <p><b>(в сотнях тысяч)</b></p> </td> </tr> <tr> <td> <p><i>Ухо</i></p> </td> <td> <p>35</p> </td> </tr> <tr> <td> <p><i>Подкова</i></p> </td> <td> <p>25</p> </td> </tr> <tr> <td> <p><i>Наковальня</i></p> </td> <td> <p>40</p> </td> </tr> <tr> <td> <p><i>Подкова &amp; Наковальня</i></p> </td> <td> <p>24</p> </td> </tr> <tr> <td> <p><i>Ухо &amp; Наковальня</i></p> </td> <td> <p>8</p> </td> </tr> <tr> <td> <p><i>Ухо &amp; Подкова</i></p> </td> <td> <p>0</p> </td> </tr> </table>
Какое количество страниц (в сотнях тысяч) будет найдено по запросу * 
Ухо | Подкова | Наковальня*?

Считается, что все запросы выполнялись практически одновременно, так что набор страниц, содержащих все искомые слова, не изменялся за время выполнения запросов.
