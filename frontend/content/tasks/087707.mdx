В программе используется одномерный целочисленный массив A 
с индексами от 0 до 10. Ниже представлен записанный на разных языках программирования фрагмент этой программы.

<u>кц</u></p> </td> <td> <p>s := 0;</p> <p>n := 10;</p> <p>for i:=0 to n-1 do begin</p> <p>  s := s + A[i] - A[i+1];</p> <p>end;</p> </td> </tr> <tr> <td colspan="2"> <p><b>Си</b></p> </td> </tr> <tr> <td colspan="2"> <p>s = 0;</p> <p>n=10;</p> <p>for (i = 0; i &lt; n; i++)</p> <p>  s = s + A[i] - A[i+1];</p> </td> </tr> </table>
В начале выполнения этого фрагмента в массиве находились числа 10, 9, 8, 7, 6, 2, 3, 4, 5, 1, 0, т.е. A[0] = 10, A[1] = 9 и т.д. Чему будет равно значение переменной *s* после выполнения данного фрагмента программы?
