<table><tr><td> <table> <tr> <td colspan="2"> <p>Дана последовательность из <i>N</i> натуральных чисел. Рассматриваются все её непрерывные подпоследовательности, такие что сумма элементов каждой из них кратна <i>k</i> = 79. Найдите среди них подпоследовательность с максимальной суммой, определите её длину. Если таких подпоследовательностей найдено несколько, в ответе укажите количество элементов самой короткой из них.</p> <p><i>Входные данные</i></p> <p>Даны два входных файла (файл <i>A</i> и файл <i>B</i>), каждый из которых содержит в первой строке количество чисел <i>N</i> (1 ≤ <i>N</i> ≤ 10 000 000). Каждая из следующих <i>N</i> строк содержит одно натуральное число, не превышающее 10 000.</p> <p>Пример организации исходных данных во входном файле:</p> <p>7</p> <p>1</p> <p>3</p> <p>4</p> <p>93</p> <p>8</p> <p>5</p> <p>95</p> <p>Для указанных входных данных при <i>k</i> = 50 искомая длина последовательности равна 2.</p> <p>В ответе укажите два числа: значение длины искомой подпоследовательности сначала для файла <i>А</i>, затем для файла <i>B</i>.</p> <p><b>Предупреждение:</b> для обработки файла <i>B</i> <b>не следует</b> использовать переборный алгоритм для всех возможных вариантов, поскольку написанная по такому алгоритму программа будет выполняться слишком долго.</p> </td> </tr> </table> </td></tr></table>
