Виталий фотографирует интересные места и события с помощью своего смартфона. Каждая фотография представляет собой растровое изображение размером 1280×1024 пикселей, при этом используется палитра из $2^{22}$ цветов. В конце дня Виталий отправляет снимки друзьям с помощью приложения-мессенджера. Для экономии трафика приложение оцифровывает снимки повторно, используя размер 1024×768 пикселей и глубину цвета 21 бит. Сколько Кбайт трафика экономится при передаче 110 фотографий?

В ответе укажите целую часть полученного числа.
