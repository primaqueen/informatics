<table><tr><td> <table> <tr> <td colspan="2"> <p>Пусть <i>S</i> – последовательность из <i>N</i> целых чисел, пронумерованных подряд начиная с 1. Обозначим <i>$S_{i}$</i>,<i> $S_{j}$,</i> <i>$S_{k}$</i> три элемента последовательности <i>S</i>, где <i>i </i>&lt; <i>j </i>&lt; <i>k</i>.</p> <p>Определите в последовательности <i>S</i> три таких числа <i>$S_{i}$</i>,<i> $S_{j}$</i>, <i>$S_{k}$</i>, что <i>S</i><i>$_{i}$ &lt; $S_{j}$</i>,<i> $S_{k}$ &lt;</i> <i>S</i><i>$_{j}$</i> и значение выражения (<i>$S_{j}$ – $S_{i}$</i>)<i> + </i>(<i>$S_{j}$ – $S_{k}$</i>) максимально. В ответе укажите найденное максимальное значение выражения (<i>$S_{j}$ – $S_{i}$</i>)<i> + </i>(<i>$S_{j}$ – $S_{k}$</i>). Гарантируется, что в последовательности есть три числа <i>$S_{i}$</i>,<i> $S_{j}$</i>, <i>$S_{k}$</i>, удовлетворяющие условию задачи.</p> <p><i>Входные данные</i></p> <p>Дано два входных файла (файл <i>A</i> и файл <i>B</i>), каждый из которых
в первой строке содержит число <i>N</i> (5 ≤ <i>N</i> ≤ 10 000 000) – количество целых чисел. Каждая из следующих <i>N</i> строк содержит одно целое число, значение которого по модулю не превышает 1000.</p> <p>В ответе укажите два числа: сначала значение искомой величины для файла <i>А</i>, затем – для файла <i>B</i>.</p> <p><i>Типовой пример организации данных во входном файле</i></p> <p><i>9</i></p> <p><i>30</i></p> <p><i>3</i></p> <p><i>7</i></p> <p><i>8</i></p> <p><i>2</i></p> <p><i>6</i></p> <p><i>1</i></p> <p><i>20</i></p> <p><i>21</i></p> <p><b></b></p> <p><i>При таких входных данных искомую максимальную сумму разностей образуют второй, четвёртый и седьмой элементы данной последовательности. Значение этой суммы разностей равно </i>(<i>8 – 3</i>)<i> + </i>(<i>8 – 1</i>)<i> = 12. Для седьмого, восьмого и девятого элементов последовательности искомая величина равна 18, но девятый элемент больше восьмого, что не удовлетворяет условию задачи. Ответом является число 12.</i></p> <p></p> <p><b>Типовой пример имеет иллюстративный характер. Для выполнения задания используйте данные из прилагаемых файлов.</b></p> <p><b>Предупреждение:</b> для обработки файла <i>B</i> <b>не следует</b> использовать переборный алгоритм, вычисляющий сумму для всех возможных вариантов, поскольку написанная по такому алгоритму программа будет выполняться слишком долго.</p> </td> </tr> </table> </td></tr></table>
