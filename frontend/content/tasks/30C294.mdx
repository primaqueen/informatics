<table><tr><td> <table> <tr> <td> <p>У медицинской компании есть <i>N </i>пунктов приёма биоматериалов на анализ. Все пункты расположены вдоль автомагистрали и имеют номера, соответствующие расстоянию от нулевой отметки до конкретного пункта. Известно количество пробирок, которое ежедневно принимают в каждом из пунктов.</p> <p>Компания планирует открыть лабораторию в одном из имеющихся пунктов. Перевозить биоматериалы разрешается на расстояние не более <i>M</i>. Пробирки перевозят в специальных транспортировочных контейнерах вместимостью не более 30 штук. Каждый транспортировочный контейнер используется для доставки пробирок только из одного пункта приёма, при этом из каждого пункта приёма может быть доставлено не более одного контейнера 
с неполной загрузкой. Пункт для лаборатории выбрали таким образом, чтобы количество доставляемых туда контейнеров 
с пробирками было максимальным. Определите необходимое количество контейнеров для доставки пробирок в эту лабораторию.</p> <p><i>Входные данные</i></p> <p>Даны два входных файла (файл <i>A</i> и файл <i>B</i>), каждый из которых 
в первой строке содержит два числа <i>N</i> и <i>M </i>(1 ≤ <i>N</i> ≤ 10 000 000, 1 ≤ <i>M</i> ≤ 10 000 000) – количество пунктов приёма биоматериалов и максимальное расстояние, на которое разрешено перевозить биоматериалы. В каждой из следующих <i>N</i> строк находятся два числа: номер пункта и количество пробирок, принимаемых на этом пункте за сутки (все числа натуральные, количество пробирок 
в каждом пункте не превышает 1000). Пункты перечислены 
в порядке их расположения вдоль автомагистрали, считая от нулевой отметки.</p> <p>В ответе укажите два числа: сначала значение искомой величины для файла <i>А</i>, затем – для файла <i>B</i>. </p> <p><i>Типовой пример организации данных во входном файле</i></p> <p><i>6 3</i></p> <p><i>1 100</i></p> <p><i>3 200</i></p> <p><i>6 4</i></p> <p><i>7 3</i></p> <p><i>8 2</i></p> <p><i>10 195</i></p> <p><i>При таких исходных данных и вместимости транспортировочного контейнера, составляющей 96 пробирок, компании выгодно открыть лабораторию в пункте 3. В этом случае количество контейнеров в ней составит: 2 + 3 + 1.</i></p> <p><b>Типовой пример имеет иллюстративный характер.Для выполнения задания используйте данные из прилагаемых файлов.</b></p> <p><b>Предупреждение:</b> для обработки файла <i>B</i> <b>не следует</b> использовать переборный алгоритм, вычисляющий сумму для всех возможных вариантов, поскольку написанная по такому алгоритму программа будет выполняться слишком долго.</p> </td> </tr> </table> </td></tr></table>
