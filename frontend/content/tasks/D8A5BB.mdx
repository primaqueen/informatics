<table><tr><td> <table> <tr> <td colspan="2"> <p>Входной файл содержит сведения о массе грузов, поступивших в транспортную компанию, и о параметрах контейнеров, которые у неё имеются. В один контейнер может быть упакован только один груз. Найдите способ для распределения максимального количества грузов по контейнерам. Если способов несколько, то нужно выбрать такой, чтобы можно было упаковать наиболее тяжёлый груз. </p> <p><i>Входные данные</i></p> <p>В первой строке входного файла находятся два натуральных числа <i>N</i> (<i>N </i>≤ 1000) и <i>M</i> (<i>M </i>≤ 1000) – количество грузов и количество контейнеров соответственно. Следующие <i>N</i> строк содержат числа, обозначающие массы грузов, затем идут <i>M</i> строк, где указана максимально допустимая масса груза для размещения в конкретном контейнере. Числа <i>M</i> и <i>N</i> могут быть не равны.</p> <p>Запишите в ответе два натуральных числа: сначала максимальное количество грузов, которое может быть упаковано, затем массу самого тяжёлого упакованного груза в этом случае.</p> <p><i>Типовой пример организации данных во входном файле</i></p> <p><i>5 6</i></p> <p><i>160</i></p> <p><i>130</i></p> <p><i>120</i></p> <p><i>150</i></p> <p><i>100</i></p> <p><i>150</i></p> <p><i>50</i></p> <p><i>155</i></p> <p><i>99</i></p> <p><i>100</i></p> <p><i>170</i></p> <p><i>При таких исходных данных максимальное количество грузов, которые могут быть упакованы в контейнеры, равно 4. При этом масса самого тяжёлого груза составит 160, а упакованными окажутся грузы массой, например, 160, 130, 120 и 100 – 
в контейнеры, выдерживающие массу 170, 150, 155 и 100.</i></p> <p></p> <p><b>Типовой пример имеет иллюстративный характер.Для выполнения задания используйте данные из прилагаемых файлов.</b></p> </td> </tr> </table> </td></tr></table>
