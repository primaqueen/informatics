<p>По каналу связи передаётся последовательность целых неотрицательных чисел – показания прибора, полученные 
с интервалом в 1 мин. в течение <em>T</em> мин. (<em>T</em> – целое число). Прибор измеряет количество атмосферных осадков, полученное регистратором за минуту, предшествующую моменту регистрации, и передаёт это значение в условных единицах измерения.</p>
<p>Определите два таких переданных числа, чтобы между моментами их передачи прошло <strong>не менее</strong> <em>K</em> мин., а их сумма была максимально возможной. Укажите найденное суммарное количество осадков.</p>
<p><em>Входные данные</em></p>
<p>Даны два входных файла (файл <em>A</em> и файл <em>B</em>), каждый из которых 
в первой строке содержит натуральное число <em>K – </em>количество минут, которое должно пройти между двумя передачами показаний, а во второй – количество переданных показаний <em>N</em> (1 ≤ <em>N</em> ≤ 10 000 000, 
<em>N </em>&gt; <em>K</em>). В каждой из следующих <em>N</em> строк находится одно целое неотрицательное число, не превышающее 100 000, обозначающее количество осадков за соответствующую минуту.</p>
<p>Запишите в ответе два числа: сначала значение искомой величины для файла <em>А</em>, затем – для файла <em>B</em>.</p>
<p><em>Типовой пример организации данных во входном файле</em></p>
<p><em>3</em></p>
<p><em>5</em></p>
<p><em>15</em></p>
<p><em>10</em></p>
<p><em>200</em></p>
<p><em>0</em></p>
<p><em>30</em></p>
<p><em>При таких исходных данных максимально возможное суммарное количество осадков равно 45 – это сумма осадков, выпавших на первой и пятой минутах.</em></p>
<p><strong>Типовой пример имеет иллюстративный характер. Для выполнения задания используйте данные из прилагаемых файлов.</strong></p>
<p><strong>Предупреждение:</strong> для обработки файла <em>B</em> <strong>не следует</strong> использовать переборный алгоритм, вычисляющий сумму для всех возможных вариантов, поскольку написанная по такому алгоритму программа будет выполняться слишком долго.</p>
