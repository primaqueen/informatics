<table><tr><td> <table> <tr> <td colspan="2"> <p>Пусть <i>S</i> – последовательность из <i>N</i> целых чисел, пронумерованных подряд начиная с 1. Обозначим <i>S</i>(<i>L</i>, <i>R</i>) подпоследовательность, состоящую из идущих подряд элементов, входящих в <i>S</i>, начиная с элемента с номером <i>L</i> и заканчивая элементом с номером <i>R</i>включительно. </p> <p>Требуется найти такую подпоследовательность <i>S</i>(<i>L</i>,<i> R</i>) максимальной длины, что сумма её элементов отрицательна и нечётна. Гарантируется, что хотя бы одна подпоследовательность требуемого вида существует.</p> <p>В ответе укажите длину искомой подпоследовательности.</p> <p><i>Входные данные</i></p> <p>Дано два входных файла (файл <i>A</i> и файл <i>B</i>), каждый из которых в первой строке содержит число <i>N</i> (5 ≤ <i>N</i> ≤ 10 000 000) – количество целых чисел. Каждая из следующих <i>N</i> строк содержит одно целое число, значение которого по модулю не превышает 1000. В ответе укажите два числа: сначала значение искомой величины для файла <i>А</i>, затем – для файла <i>B</i>.</p> <p><i>Типовой пример организации данных во входном файле</i></p> <p><i>6</i></p> <p><i>20</i></p> <p><i>2</i></p> <p><i>–8</i></p> <p><i>4</i></p> <p><i>–5</i></p> <p><i>3</i></p> <p><i>При таких входных данных </i><i>L</i><i> = 2, </i><i>R</i><i> = 5. Сумма подпоследовательности равна (2 + (–8) + 4 + (–5)) = –7</i>. <i>Ответом является длина этой подпоследовательности, равная 4.</i></p> <p></p> <p><b>Типовой пример имеет иллюстративный характер. Для выполнения задания используйте данные из прилагаемых файлов.</b></p> <p><b>Предупреждение:</b> для обработки файла <i>B</i> <b>не следует</b> использовать переборный алгоритм, вычисляющий сумму для всех возможных вариантов, поскольку написанная по такому алгоритму программа будет выполняться слишком долго.</p> </td> </tr> </table> </td></tr></table>
