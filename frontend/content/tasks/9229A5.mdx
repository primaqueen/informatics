Исполнитель преобразует число на экране.

У исполнителя есть три команды, которые обозначены латинскими буквами:

***A*. Прибавить 1**

***B*. Прибавить 3**

***C*. Умножить на 3**

Программа для исполнителя – это последовательность команд.

Сколько существует программ, для которых при исходном числе 2 результатом является число 20, при этом траектория вычислений содержит число 16 и не содержит 12?

Траектория вычислений программы – это последовательность результатов выполнения всех команд программы. *Например*, для программы ***CBA* **при исходном числе 7 траектория состоит из чисел 21, 24, 25.
