<table><tr><td> <table> <tr> <td colspan="2"> <p>Пусть <i>S</i> – последовательность из <i>N</i> целых чисел, пронумерованных подряд начиная с 1. Обозначим <i>S</i>(<i>L</i>, <i>R</i>) подпоследовательность, состоящую из идущих подряд элементов, входящих в <i>S</i>, начиная с элемента с номером <i>L</i> и заканчивая элементом с номером <i>R</i>. </p> <p>Требуется найти такие значения номеров элементов <i>L</i>,<i>M</i>,<i>R</i>,где 
0 &lt; <i>L &lt; </i><i>M &lt; </i><i>R </i>– 1 (т.е. между элементами с номерами <i>M</i> и <i>R </i>есть ещё как минимум один элемент),чтобыразность суммы элементов подпоследовательности <i>S</i>(<i>M</i> + 1, <i>R</i>) и суммы элементов подпоследовательности <i>S</i>(<i>L</i>,<i>M</i>) была максимальна. </p> <p>В ответе укажите максимальное значение разности подобных сумм.</p> <p><i>Входные данные</i></p> <p>Дано два входных файла (файл <i>A</i> и файл <i>B</i>), каждый из которых в первой строке содержит число <i>N</i> (5 ≤ <i>N</i> ≤ 10 000 000) – количество целых чисел. Каждая из следующих <i>N</i> строк содержит одно целое число, значение которого по модулю не превышает 1000.</p> <p>В ответе укажите два числа: сначала значение искомой величины для файла <i>А</i>, затем – для файла <i>B</i>.</p> <p><i>Типовой пример организации данных во входном файле</i></p> <p><i>7</i></p> <p><i>20</i></p> <p><i>4</i></p> <p><i>–2</i></p> <p><i>13</i></p> <p><i>–1</i></p> <p><i>2</i></p> <p><i>–10</i></p> <p><b></b></p> <p><i>При таких входных данных </i><i>L = 2, </i><i>M = 3, </i><i>R = 6. Искомая максимальная разность равна (13 + (–1) + 2) – (4 + (–2)) = 12.</i> <i>Подпоследовательность</i> <i>«–2 13 –1»</i> <i>разбить на две подпоследовательности требуемого вида невозможно. </i></p> <p><i>Ответом является число 12.</i></p> <p></p> <p><b>Типовой пример имеет иллюстративный характер. Для выполнения задания используйте данные из прилагаемых файлов.</b></p> <p><b>Предупреждение:</b> для обработки файла <i>B</i> <b>не следует</b> использовать переборный алгоритм, вычисляющий разность для всех возможных вариантов, поскольку написанная по такому алгоритму программа будет выполняться слишком долго.</p> </td> </tr> </table> </td></tr></table>
