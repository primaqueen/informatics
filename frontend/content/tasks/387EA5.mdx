Сколько существует различных наборов значений логических переменных 
$x_{1}$, $x_{2}$, … $x_{6}$, $y_{1}$, $y_{2}$, … $y_{6}$, которые удовлетворяют всем перечисленным ниже условиям?

$x_{1}$ → $y_{1} = 1$

($x_{2}$ → ($x_{1}$ /\ $y_{2}$)) /\ ($y_{2}$ → $y_{1}$) = 1

($x_{3}$ → ($x_{2}$ /\ $y_{3}$)) /\ ($y_{3}$ → $y_{2}$) = 1

…

($x_{6}$ → ($x_{5}$ /\ $y_{6}$)) /\ ($y_{6}$ → $y_{5}$) = 1

В ответе **не нужно** перечислять все различные наборы значений переменных $x_{1}$, $x_{2}$, … $x_{6}$, $y_{1}$, $y_{2}$, … $y_{6}$, при которых выполнена данная система равенств. 
В качестве ответа Вам нужно указать количество таких наборов.
