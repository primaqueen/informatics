Исполнитель преобразует число, записанное на экране.

У исполнителя есть три команды, которым присвоены номера:

**1. Прибавить 1**

**2. Прибавить 2**

**3. Умножить на 2**

Первая из них увеличивает число на экране на 1, вторая увеличивает его на 2, третья умножает на 2.

Программа для исполнителя – это последовательность команд.

Сколько существует таких программ, которые преобразуют исходное число 3 в число 13 и при этом траектория вычислений программы содержит числа 10 и 12? Траектория должна содержать оба указанных числа.

Траектория вычислений программы – это последовательность результатов выполнения всех команд программы. Например, для программы **132 **при исходном числе 7 траектория будет состоять из чисел 8, 16, 18.
