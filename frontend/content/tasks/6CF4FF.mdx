Исполнитель преобразует число на экране.

У исполнителя есть две команды, которые обозначены латинскими буквами:

***A*. Вычти 2**

***B*. Найди целую часть от деления на 2**

Программа для исполнителя – это последовательность команд.

Сколько существует программ, для которых при исходном числе 36 результатом является число 2, и при этом траектория вычислений содержит число 8?

Траектория вычислений программы – это последовательность результатов выполнения всех команд программы.

*Например,* для программы ***ABB***при исходном числе 13 траектория состоит из чисел 11, 5, 2.
