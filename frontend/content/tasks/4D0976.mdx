На вход алгоритма подаётся натуральное число $N$. Алгоритм строит по нему новое число $R$ следующим образом.

1. Строится двоичная запись числа $N$.

2. Далее эта запись обрабатывается по следующему правилу:

   а) если число чётное, то к двоичной записи числа слева дописывается $10$;

   б) если число нечётное, то к двоичной записи числа слева дописывается $1$ и справа дописывается $01$.

Полученная таким образом запись является двоичной записью искомого числа $R$.

*Например,* для исходного числа $4_{10} = 100_{2}$ результатом будет являться число $20_{10} = 10100_{2}$, а для исходного числа 
$5_{10} = 101_{2}$ результатом будет являться число $53_{10} = 110101_{2}$.

Укажите **минимальное** число $N$, после обработки которого с помощью этого алгоритма получается число $R$, большее, чем $441$. В ответе запишите это число в десятичной системе счисления.
