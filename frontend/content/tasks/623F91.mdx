<table><tr><td> <table> <tr> <td colspan="2"> <p>Отдел маркетинга сети продуктовых магазинов составляет рейтинг продуктов по информации об их сроках хранения с момента изготовления и после вскрытия упаковки. Для каждого продукта известен срок его хранения с момента изготовления и срок годности 
к употреблению после вскрытия упаковки. Продукты пронумерованы начиная с единицы.</p> <p>В рейтинговом списке маркетологи располагают продукты 
по следующему алгоритму:</p> <p>– все 2<i>N</i> чисел, обозначающих срок хранения и срок годности 
к употреблению для <i>N</i> продуктов, упорядочивают по возрастанию;</p> <p>– если минимальное число в этом упорядоченном списке – срок хранения, то продукт в рейтинге занимает первое свободное место от его начала;</p> <p>– если минимальное число – срок годности к употреблению, то продукт занимает первое свободное место от конца рейтинга;</p> <p>– если число обозначает срок хранения или срок годности 
к употреблению уже рассмотренного продукта, то его не принимают во внимание. </p> <p>Этот алгоритм применяется последовательно для размещения всех 
<i>N </i>продуктов.</p> <p>Определите номер последнего продукта, для которого будет определено его место в рейтинге, и количество продуктов, которые займут в рейтинге более низкие места. </p> <p><i>Входные данные</i></p> <p>В первой строке входного файла находится натуральное число <i>N</i> (<i>N </i>≤ 1000) – количество продуктов. Следующие <i>N</i> строк содержат пары чисел, обозначающих соответственно срок хранения продукта с момента изготовления и срок годности к употреблению после вскрытия упаковки (все числа натуральные, различные).</p> <p>Запишите в ответе два натуральных числа: сначала номер последнего продукта, для которого будет определено его место 
в рейтинге, затем – количество продуктов, которые займут 
в рейтинге более низкие места.</p> <p></p> <p><i>Типовой пример организации данных во входном файле</i></p> <p><i>5 </i></p> <p><i>30 50</i></p> <p><i>100 155</i></p> <p><i>150 170</i></p> <p><i>10 160</i></p> <p><i>120 55</i></p> <p><i>При таких исходных данных порядок расположения продуктов 
в рейтинге следующий: 4, 1, 2, 3, 5. Последним займёт своё место 
в рейтинге продукт 3. При этом один продукт </i>зай<i>мёт в рейтинге более низкое место. </i></p> <p><b>Типовой пример имеет иллюстративный характер.Для выполнения задания используйте данные из прилагаемых файлов.</b></p> </td> </tr> </table> </td></tr></table>
