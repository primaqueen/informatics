Какой минимальный объём памяти (в Кбайт) нужно зарезервировать, чтобы можно было сохранить любое растровое изображение размером
640×320 пикселей при условии, что в изображении могут использоваться 
64 различных цвета? Для кодирования цвета каждого пикселя используется одинаковое количество бит, коды пикселей записываются в файл один 
за другим без промежутков. Искомый объём не учитывает размера заголовка файла. В ответе запишите только целое число, единицу измерения писать 
не нужно.
