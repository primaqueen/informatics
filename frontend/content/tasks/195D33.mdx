Дано целое положительное число *N*, не превосходящее 1000. Нужно написать программу, которая определяет, является ли это число степенью числа 7: выводит на экран либо такое целое число *K*, что 7*$^{K}$* = *N*, либо сообщение «NO», если такого числа не существует.

им программа для Вашего удобства приведена на пяти языках программирования.
