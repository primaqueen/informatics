Виталий фотографирует интересные места и события с помощью своего смартфона. Каждая фотография представляет собой растровое изображение размером 2560×1440 пикселей, при этом используется палитра из $2^{23}$ цветов. В конце дня Виталий отправляет снимки друзьям с помощью приложения-мессенджера. Для экономии трафика приложение оцифровывает снимки повторно, используя размер 1920×1080 пикселей и глубину цвета 22 бит. Сколько Кбайт трафика экономится при передаче 130 фотографий?

В ответе укажите целую часть полученного числа.
