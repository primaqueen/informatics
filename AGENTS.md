## Регламент проверок

- После любых правок запускай `uv run ruff check .`.
- Дополнительно проверяй типы: `uv run mypy parse_fipi_pages.py`.
- Для применения автоформатирования используй `uv run ruff format .` (опционально перед коммитом).
- Для актуализации окружения — `uv sync`.
- Для проверки сборки данных — `uv run python parse_fipi_pages.py`.
